# Mail Bruteforce
Script for email bruteforce and that send HTML messages.

### Bruteforce

### Email Sender
Send Computer Informations, IPs, HTML messages and files

## Installation 
_Python_ - `3.8`
```
git clone https://github.com/vLeeH/MailSender.git
cd MailSender
pip install smtplib
python Mail/mail.py
```
